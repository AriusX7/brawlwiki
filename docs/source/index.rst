Welcome to Brawl Wiki's documentation!
======================================

- This library is a sync wrapper for https://ariusx7.github.io/brawlwiki-api/ (BrawlWiki API).
- Python 3.6 or later is required.

Features
~~~~~~~~

- Covers all of the endpoints of the API.
- Access constant values such as attack name, description, etc!
- Easy to use with an object oriented design.
